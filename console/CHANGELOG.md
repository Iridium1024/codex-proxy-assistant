# Changelog

## 0.1.3 - 2026-07-18

- 将图形版本的核心功能迁移为独立的“CMD 启动器 + PowerShell 中文菜单”。
- 无参数启动默认进入只读菜单，不再直接执行持久修复。
- 保留自动检测、真实 HTTPS 代理验证、修改预览、持久修复、配置快照与恢复。
- 保留临时 CLI 代理试运行，并将实现迁移到 PowerShell 核心。
- 保留用户主动确认后执行的最小 Codex 真实连接验证。
- 去除 Python、PyQt5、PyInstaller 和 EXE 运行时依赖。
- 新增独立发布打包脚本；ZIP 解压后的第一层即可看到“启动修复.cmd”。
- 作为兼顾易用性、兼容性和体积的默认推荐下载版本正式发布。
